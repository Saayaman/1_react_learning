## 1. まずはプロジェクトを作成してコミットしよう！

### Npmの場合

```
$ npx create-react-app my-app
$ cd my-app
$ npm start
```

*npmがない人はinstallしてください　
https://nodejs.org/en/download/package-manager/

### yarn の場合

```
$ npx create-react-app my-app
$ cd my-app
$ yarn start
```

*Yarnがない人はinstallしてください　
https://yarnpkg.com/en/docs/install#mac-stable
